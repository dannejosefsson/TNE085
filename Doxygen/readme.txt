To generate the html API do the following:

1. Install Doxygen from www.doxygen.org (version 1.8.2 was used during this project) in Windows

2. Start the Doxywizard

3. Open the doxyfile Doxyfile_TNE085 in the Doxywizard (File > Open)

4. Specify the working directory from which the doxygen will run at the top of the first page.
   This directory is the bin folder (doxygen/bin).

5. Under the tab Wizard choose the topic Project and specify
   * Source code directory and (This is where the source code is located)
   * Destination directory     (This is where the html API will be placed)

6. Run Doxygen
